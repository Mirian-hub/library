# Icon directive

Used for displaying an icon. This is a directive.

## When to use

Whenever there is a need for a visual indicator that describes an operation - the eye icon for a search operation, thrash bin for delete etc. This being a directive,
you can use it on **span** and **i** elements.

## How to use

Before using, make sure you have imported **UpIconsModule**

    import { UpIconsModule } from 'community-design-system';

Start using the component

    <i upIcon='search_medium'
       color='var(up-brand-700)'
       width='32'
       height='32'
       fallback='fallback_icon'>
    </i>

**If no color is provided, the directive will default to var(--up-brand-1). Consult the Colors section for more information.**

## Examples

    <i upIcon='search_medium'
       color='var(up-brand-700)'
       width='32'
       height='32'
       fallback='fallback_icon'>
    </i>
