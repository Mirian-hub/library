# Country Flag

Used for displaying the flag of a country. This is using the icon directive.

## When to use

Whenever there is a need for a visual indicator of the country using the country flag.

## How to use

Before using, make sure you have imported **UpIconsModule**

    import { UpIconsModule } from 'community-design-system';

Start using the component

    <i upIcon='ro'></i>

The icon name to be used is the [iso 3166](https://en.wikipedia.org/wiki/List_of_ISO_3166_country_codes) code for that country

Please check the [documentation](?path=/docs/icon-icon--icon) for the icon directive to see all the available parameters.
