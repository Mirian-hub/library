# UI Icons

Here are displayed all the available icons of the Design System

## How to use

Before using, make sure you have imported **UpIconsModule**

    import { UpIconsModule } from 'community-design-system';

Start using the component

    <i upIcon='accounts_payable_medium'></i>

Please check the [documentation](?path=/docs/icon-icon--icon) for the icon directive to see all the available parameters.
